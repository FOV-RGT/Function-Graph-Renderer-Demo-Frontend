## <center> 项目运行
1.前置条件
- **Node.js 16** 及以上版本
- npm包管理器

2.安装依赖
```sh
npm i
```

3.启动项目
```sh
npm run dev
```
或直接运行NPM脚本`dev`
- 启动[后端服务器](https://github.com/FOV-RGT/Function-Graph-Renderer-Demo-Backend) （v0.9.0及以上版本不再需要后端服务器支持图形渲染）
